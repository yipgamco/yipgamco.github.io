参数传递

Java中参数传递是以传值方式进行的，实参储存的值传给形参。

  1. 参数是基本数据类型(int、float)，传值
    实参存的是变量的值，传参时把变量的值传给形参。这时候形参和实参操作的不是同一储存单元，修改形参的值不会影响实参
  2. 参数是引用类型(对象、数组等)，传址
    实参和形参中存放的都是引用对象地址，传参时实参把引用地址传给形参。这时候形参和实参操作的是同一储存单元。
    也就是说，参数是基本类型时，形参和形参互不影响。参数是引用类型时，相互影响。

使用包组织类

建立包

  • java.* java核心包
  • javax.* java拓展包
  • org.* 组织扩展包
    创建包要使用关键字package，而且要放在第一行。
    每个包对应一个目录路径，以下定义意味着当前目录下有一个test子目录，该包中所有类的代码文件都存放在test目录下

引用包

  1. 使用包名作前缀
    new java.util.Date()
  2. 用import语句加载整个类
    import java.util.Date
  3. 用import语句加载整个包
    import java.util.
    (import java.util.    并不是把util整个包加载进程序中，而是告诉编译器使用这些类时到什么地方去查找类的代码)
  4. 通过设置环境变量CLASSPATH指明字节码文件路径
    使用DOS命令修改：
    SET CLASSPAT = XXXXXX
